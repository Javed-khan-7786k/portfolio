# 🌟 Professional Portfolio | Javed Alam

A premium, modern, and highly interactive developer portfolio website designed for **Javed Alam**, MERN Stack Developer. Built using React 19, TypeScript, Vite, Tailwind CSS v4, and Motion (Framer Motion).

Check out the live repository at [github.com/Javed-khan-7786k/portfolio](https://github.com/Javed-khan-7786k/portfolio).

## ✨ Features

- **📱 Premium Responsive Design**: Fully responsive layout tailored for all screens from mobile to desktop.
- **✨ Custom Interactive Cursor**: A dual-ring cursor that tracks mouse movement, morphs on clickable elements, and responds to click events.
- **🎨 Interactive Background**: A custom dynamic radial-gradient glow that tracks cursor movements in real-time.
- **⚡ Seamless Animations**: Smooth, high-performance page transitions and micro-interactions powered by Framer Motion.
- **🗂️ Content Sections**:
  - **Hero**: An eye-catching introduction banner with social links.
  - **About**: Professional summary and categorized skill listings.
  - **Resume**: Interactive timeline highlighting academic education and experience.
  - **Portfolio**: Showcase card grid of full-stack MERN projects with source code and live site links.
  - **Contact**: A functional validation-backed contact form with integrated quick links.

## 💻 Getting Started

Follow these steps to run the portfolio website locally on your computer.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed (version 18+ recommended).

### Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Javed-khan-7786k/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Set Up Environment Variables**
   Create a `.env.local` file in the root directory (you can copy `.env.example` as a template):
   ```bash
   cp .env.example .env.local
   ```
   Provide your Gemini API Key in the `.env.local` file:
   ```env
   VITE_GEMINI_API_KEY=your_gemini_api_key_here
   ```

4. **Start Development Server**
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:3000`.

5. **Build for Production**
   ```bash
   npm run build
   ```
   This command will output static build files into the `dist/` directory, ready to deploy to Netlify, Vercel, or GitHub Pages.
